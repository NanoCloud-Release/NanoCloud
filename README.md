## 快速开始

1. 打开 [NanoCloud](https://edu.yuque.men/auth/register?code=KGPt0kjX)并创建账户。
2. 根据流量、设备数量和稳定性要求选择套餐。
3. 下载官方客户端，或在 **仪表板 → 一键导入** 中选择第三方客户端。
4. 打开节点状态，根据当前网络选择“通用”或“流量”节点。
5. 绑定 [@NanoAir_bot](https://t.me/NanoAir_bot)，获取 Telegram 专用代理和每日签到流量。

## Telegram Bot

官方 Bot：[@NanoAir_bot](https://t.me/NanoAir_bot)

绑定后可以在 Telegram 中完成购买订阅、每日流量签到、获取订阅、提交使用反馈和获取 Telegram 专用代理。

### Telegram 专用代理

- 无需启动完整代理客户端即可访问 Telegram。
- 后台持续收取消息时更省电。
- 不适用于猎户座 1 元套餐。
- Telegram 可能限制免费账户的视频速度；视频卡顿时，瓶颈可能来自 Telegram 账户而非 NanoCloud 链路。

## 注意事项

- Wi-Fi、光猫和代理软件都需要开启 IPv6，才能使用 IPv6“流量”节点；也可直接使用支持 IPv6 的手机流量连接。
- 购买后通常需要等待约 5 分钟才能正常使用。
- 官方客户端能够提供更多长期有效的稳定节点。Clash Verge、V2Ray 等第三方客户端可能暴露订阅服务器，从而增加被封锁的风险。
- 套餐、节点、带宽和在线状态可能调整，请以控制台实时信息为准。
