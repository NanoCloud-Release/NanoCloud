## 注意事项
- Wifi、光猫和代理软件都开启IPv6，才能正常使用IPv6“流量”节点，或者也可直接用手机流量（IPv6）连接
- 购买后需要等待5分钟才能正常使用
- 使用官方专用代理软件能有数量更多、长期有效的稳定节点。使用clash verge、v2ray等因会泄露订阅服务器，较易被封锁。

## 快速开始

1. 打开 [NanoCloud](https://edu.yuque.men/auth/register?code=KGPt0kjX)并创建账户。
2. 根据流量、设备数量和稳定性要求选择套餐。
3. 下载官方客户端，或在 **仪表板 → 一键导入** 中选择第三方客户端。
4. 打开节点状态，根据当前网络选择“通用”或“流量”节点。
5. 绑定 [@NanoAir_bot](https://t.me/NanoAir_bot)，用于获取tg代理和领取签到流量。

## Telegram 专用代理
无需启动完整代理客户端即可访问 Telegram。后台持续收取消息时更省电。
不适用猎户座1元套餐。
Telegram 会限制部分免费账户的视频速度；经常播放视频时卡顿，瓶颈可能来自 Telegram 账户而不是 NanoCloud 链路。
