## 这轮变化影响了什么

### 中转源头受限

过去一台中转机器被封，服务商可能只需更换 IP。4 月的运维通报描述了另一种风险：被清退的是上游带宽或整个机房，受影响的不只是一个节点。中转仍可能比普通直连快，但它增加了国内入口这个集中故障点。

### 直连一直是低价备用

直连不依赖国内中转资源，封锁期间可能比失效的中转更容易恢复；代价是普通公网路由更受运营商、地区、晚高峰和封锁影响。直连适合作为低价方案或第二通路，不等于天然稳定。

### 专用客户端开始承担节点保护

一些机场不再把完整订阅直接交给第三方客户端，而在自己的客户端内下发配置。官方客户端通常可获取更多节点，开源客户端可获取的节点较少。需要把配置接入 Clash 等开源客户端时，可尝试 [Sub2Clash](https://github.com/0xWans/Sub2Clash) 进行本地转换和管理配置。

### 老牌和高价都不能保证未来

快连长期运营、用户规模大、客户端成熟，仍然停止中国大陆业务。另一方面，奶昔只上调 5%，论坛里也有人表示自己的机场没有涨价。正确结论不是“机场都会跑路或暴涨”，而是运营历史、价格和当前可用性必须分开判断。

## 现在更合理的购买顺序

1. 先明确最重要的场景：AI、流媒体、晚高峰、固定 IP、大流量或低价。
2. 查看当前公告、节点状态、客户端和退款规则，不只看推荐文章。
3. 优先试用或月付；连续使用后再考虑更长周期。
4. 用自己的城市、运营商和常用设备测试，不用他人的测速图替代。
5. 工作刚需准备不同入口或不同技术路线的备用服务。
