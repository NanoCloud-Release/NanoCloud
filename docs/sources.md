## 更新规则

1. 套餐和节点数据以控制台为准，并标注核对日期。
2. 外部服务状态查一手公告，不从搜索摘要直接下结论。
3. 单次测速不写成长期保证。
4. AFF 和普通注册链接同时公开。
5. 论坛内容标注来源类型与证据边界。
6. 已过时信息删除或明确标注时间。
